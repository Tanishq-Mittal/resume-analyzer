# Resume Analyzer App

This project analyzes resumes and gives feedback to improve job readiness.

## Features
- Resume PDF upload
- Skill extraction
- Resume score out of 10
- Missing skills detection
- Suggestions for improvement

## Tech Stack
Python
Flask
HTML

## How to Run
pip install -r requirements.txt
python app.py

Open browser:
http://127.0.0.1:5000
